This site is under construction.

This website aims to replace https://github.com/interledgerjs/ilp-kit/wiki/Known-ILP-Nodes.

We have plans for adding uptime statistics, but for now, it just gives a simple list of known ILP nodes. Diagnostics like 'Ping' are periodically updated by a script.

Known ILP nodes

This is a list of known nodes as submitted by the people running them. For now, a node is assumed to run a connector as well as a ledger, in the way ilp-kit does. You can add your ILP node by opening a pull request.

See also the performance graphs (still experimental)
